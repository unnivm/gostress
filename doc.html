GoStress is a lightweight and fast stress testing tool written in Go. It helps you test the performance, latency, and reliability of HTTP endpoints under load.

Download the pre-built binary for your OS from the main page:

  • Linux (amd64): gostress-linux

  • macOS (arm64): gostress.dmg

Make it executable (Linux/macOS):

chmod +x gostress-linux

gostress --url <URL> --c <CONCURRENCY> --d <DURATION>

Example:

This sends HTTP GET requests to https://example.com using 50 concurrent workers for 30 seconds.
